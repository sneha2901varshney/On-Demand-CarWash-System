# On-Demand-CarWash-System
Built an On-Demand Car Wash platform using Spring Boot microservices architecture exposing REST/JSON APIs with Angular-based UI.  
Designed services for user management, booking, payments, reviews, and admin operations with independent databases (H2/MySQL/MongoDB). 
Implemented RabbitMQ messaging, logging, exception handling, and unit testing to ensure scalable and reliable service communication.
